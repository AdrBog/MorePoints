If you see this page, you have (probably) successfully setup More Points

How to access a point

To access a point, enter the address /point/POINT_NAME in the URL bar

If you have not configured any point yet, you can access the point that comes by default with the More Points installation:

  • URL: /point/sample
  • Password: 1234
How to create a point

Access the admin page, the default password is admin, if you want to change it, edit the config.json file stored in the config folder

Remember that the changes will not be applied until you restart the FTP server (server.py).
