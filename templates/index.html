How to access a site

To access a site, enter the address /site/SITE_NAME in the URL bar

If you have not configured any site yet, you can access the site that comes by default with the More Points installation:

  • URL: /site/sample
  • Password: 1234
How to create a site

First you need a place on the server to save the files.
To keep your sites organized I recommend using the sites folder that comes with the installation.

Create the folder SITE_NAME in the sites folder, once you have created the route, you will have to configure the site

Sites are configured in config/sites.json

To create a site, add a section in the file, here is an example of a configured site:

    "SITE_NAME": {
        "password": "1234",
        "perm": "elradfmwMT",
        "root": "sites/SITE_NAME"
    }
  • password: Do i need to explain?
  • perm: Site permissions, click here to get info about permissions
    If you have no idea, just set elradfmwMT for read and write permissions or elr for read permissions only.
  • root: The location on the server where the files will be stored, as mentioned above, I recommend using the sites directory

Additionally you can add extra permissions for specific directories of the site

    "SITE_NAME": {
        "password": "1234",
        "perm": "elr",
        "root": "sites/SITE_NAME",
        "extra_perm": {
            "/folder": "elradfmwMT"
        }
    }

Remember that the changes will not be applied until you restart the FTP server (server.py).

Once you're ready go to /site/SITE_NAME and enter the password
